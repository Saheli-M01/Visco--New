**Visco**  

Visco is a React-based web application built to help students and developers understand how algorithms work through real-time visualizations, step-by-step animations, and syntax-highlighted inbuilt code snippets. Whether you're learning your first sorting algorithm or brushing up on graph traversals, Visco makes it visual.

__✨ Features__  

🔁 Dynamic Algorithm Animations

💻 Syntax Highlighted Code

⚡ Interactive Controls

🎨 Clean UI with React Bootstrap & Sass


**🚀 Getting Started**  

Prerequisites
Node.js (>= 16)

npm or yarn

**Installation**  

git clone https://github.com/Saheli-M01/Visco--New.git  

cd Visco--New  

npm install  

npm run dev
